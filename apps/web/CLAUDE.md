# @storik/web

React 19 + Vite + TanStack Router (file-based) + Panda CSS + Motion.

## Routing

- File-based routes in `src/routes/`
- TanStack Router plugin auto-generates `src/routeTree.gen.ts` — **never edit**

## Styling

- Panda CSS — use `css()` and recipes from `styled-system/`
- `styled-system/` is generated by `panda codegen` — **never edit**
- Animation: use `motion` library

## Commands

- `pnpm dev:web` — Vite dev server on port 5173 (proxies /api → 3001)
- `pnpm --filter @storik/web build` — production build
- `pnpm test` — vitest from root (uses happy-dom)

## Rules

- Never import from `styled-system/` paths directly in tests — mock if needed
- Run `panda codegen` (via `pnpm prepare`) if Panda config changes
